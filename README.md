# E-Commerce System (Spring Boot & MongoDB)

A technical study on **NoSQL architectural patterns** using Spring Boot and MongoDB. This project demonstrates high-performance data modeling techniques beyond traditional RDBMS constraints.

## Core Concepts

*   **Schema-less Catalog:** Dynamic product attributes to handle diverse categories in a single collection.
*   **Data Snapshotting:** Embedding product metadata into order documents (denormalization) to ensure historical data integrity.
*   **Aggregation Pipelines:** Real-time business intelligence and reporting using MongoDB's powerful aggregation framework.

## Tech Stack

*   **Language:** Java 17
*   **Framework:** Spring Boot 3.x
*   **Database:** MongoDB
*   **Utilities:** Lombok, Spring Data MongoDB
